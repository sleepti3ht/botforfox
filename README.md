# LisSkins Monitor Bot

A lightweight Telegram bot that monitors the **lis-skins.com** marketplace for specific Dota 2 items and sends notifications when matching items appear.

## Features

- Infinite marketplace scanning  
- Custom item rules (gems, styles, max price)  
- Telegram notifications with inline buttons  
- Prevents duplicate alerts during a single runtime  
- Clean and minimal codebase
